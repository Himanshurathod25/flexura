# Flexura — Digital Growth

## Project info

This repository contains the Flexura digital growth starter application.

## How can I edit this code?

You can work locally using your preferred IDE. The only requirements are Node.js and npm.

Quick start:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Build the project and deploy the `dist` output to your hosting provider. Example:

```sh
npm run build
# then upload the `dist` folder to your host
```
